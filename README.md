# Research_Forum



## 📌 Project Overview
Research Forum is a full-stack web application that enables students and researchers to publish research articles, upload PDF documents, and collaborate through discussions. The platform includes secure authentication, role-based access control, and an admin approval workflow to ensure quality content management.

---

## 🎯 Problem Statement
Researchers and students need a centralized platform to share research work, receive feedback, and manage publications securely. This project provides a structured environment for publishing, reviewing, and interacting with research content.

---

## 📂 Features
- 👤 User Registration & Login (JWT Authentication)
- 🔐 Role-Based Access (User/Admin)
- 📝 Research Article Publishing
- 📄 PDF Upload Support
- ✅ Admin Approval Workflow
- 💬 Comments & Discussions
- 👍 Upvote & 👎 Downvote System
- 🔍 Search, Categories & Tags
- 📱 Responsive User Interface
- ⚙️ Admin Moderation Dashboard

---

## 🛠 Technologies Used
- React.js
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Multer
- REST API

---

## 🚀 Project Workflow
1. User Registration & Login
2. Create Research Post
3. Upload Research PDF
4. Submit for Admin Approval
5. Admin Reviews & Approves
6. Published Posts Visible to Users
7. Users Search, Comment & Vote
